# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Reporting a Vulnerability
If you discover a security vulnerability in BayonHub, please report
it responsibly. Do not create a public GitHub issue.

Contact us privately:
- Email: security@bayonhub.com
- Response time: within 48 hours
- We will acknowledge receipt, investigate, and patch critical issues
  within 7 days.

## Scope
- bayonhub-api (Node.js/Express backend)
- bayonhub-app (React frontend)
- Database (PostgreSQL via Prisma)

## Out of Scope
- Social engineering attacks
- Physical security
- Third-party services (Railway, Cloudinary, Telegram)

Thank you for helping keep BayonHub safe.
